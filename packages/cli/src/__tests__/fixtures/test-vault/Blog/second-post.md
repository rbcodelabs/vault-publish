# The Power of Plain Text

No frontmatter here either. Plain markdown, published via folder inclusion.

There's something freeing about writing in plain text. No vendor lock-in,
no proprietary formats — just `.md` files you can open anywhere.

## Why Obsidian

- Local-first storage
- Wikilinks between notes
- A vibrant plugin ecosystem
- Fast, even with 10,000 notes

The graph view alone is worth it.
