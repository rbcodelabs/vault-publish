# A Note That Links to Others

This note demonstrates wikilinks. It references [[Pages/about-me]] and
[[Pages/contact]] — both of which are published notes.

Linking to an unpublished note like [[draft]] is also valid Obsidian syntax,
but the resolved link on the web will show as a dead link (expected behaviour).

## Why Wikilinks Matter

They're the connective tissue of a knowledge graph. vault-publish preserves
them and builds a graph manifest from the `outlinks` so the graph view works.
