# Tagging in Obsidian

This note has frontmatter `tags` (the standard Obsidian property).
vault-publish stores these and surfaces them on the published note.

## Two Flavours of Tags

Obsidian supports tags two ways:

1. **Frontmatter** — `tags: [obsidian, writing]` — structured, parseable
2. **Inline** — `#obsidian` anywhere in the body — flexible but harder to query

Both are valid. The parser handles both.

## Tag Tips

- Use kebab-case for multi-word tags: `#knowledge-management`
- Tags help readers discover related notes
- They also filter notes in Obsidian's search and graph
